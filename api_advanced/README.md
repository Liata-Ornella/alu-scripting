 # 0x16. API advanced

## Description
This project is part of the **ALU Scripting** curriculum.  
It focuses on using the **Reddit API** to retrieve and process data such as subscribers, hot posts, and keyword counts.

You will learn how to:
- Query the Reddit API without authentication.
- Handle pagination.
- Use recursion in API calls.
- Parse JSON data.
- Sort and count results.

---

## Requirements
- Ubuntu 20.04 LTS
- Python 3.4.3 or higher
- Allowed editors: `vi`, `vim`, `emacs`
- All your files will be interpreted/compiled on Ubuntu 20.04 LTS using Python 3
- Your code should follow `pycodestyle` (PEP8) style guidelines
- All your files must be executable
- The first line of all your files should be exactly `#!/usr/bin/python3`
- All your files must end with a new line
- All your modules should have documentation (`python3 -c 'print(__import__("my_module").__doc__)'`)
