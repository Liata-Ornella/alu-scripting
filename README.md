# Reddit API Tasks

This project contains Python scripts that interact with the **Reddit API**.  
It is part of the **ALU Scripting - API Advanced** exercises.

## Files Included
| File | Description |
|------|--------------|
| `0-subs.py` | Returns the number of subscribers for a given subreddit |
| `1-top_ten.py` | Prints the top 10 hot post titles for a subreddit |
| `2-recurse.py` | Recursively lists all hot post titles for a subreddit |
| `3-count.py` | Recursively counts keyword occurrences in hot post titles |

## Usage
Each script can be run from the command line with a corresponding `main` test file.  
Example:

```bash
python3 0-main.py programming
